Our proposal

Our idea is to create a gamified productivity manager. Users will be able to set activities and tasks, allowing them to track their progress based on objectives they set themselves. When they complete tasks, they get rewarded with XP, allowing the users to level up and receive rewards for completing their tasks. Users will be able to compare their progress to others via leaderboards. The program will show a calendar of activities based on what the user inputs into the app, allowing users to easily track their activities and tasks.

Key Features

● Database tables for user accounts, calendars, and progression.
● Time tracking functionality.
● Profile pictures (default and user-uploaded)
● Login authentication
● Levelling system
● Level-up rewards
● Leaderboards
