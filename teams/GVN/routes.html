Admin Page Routes/URLs

-- User --

GET/admin/user - List all of the users/members in the admin dashboard

POST/admin/user/addUser - Uses POST method to create a new user records and adds it to database

GET/admin/user/viewUser/:userid - Lists a selected user details

GET/admin/user/updateUser/:userid - Page to update a selected user data

-- Routine --

GET/admin/routine -- List all of the routines in the admin dashboard

POST/admin/routine/addRoutine - Uses POST moethod to create new and adds it to the database

GET/admin/routine/viewUser/:routineid - Lists a selected routine data

GET/admin/routine/updateRoutine/:routineid - Page to update a selected routine data

-- Exercise --

GET/admin/exercise -- List all of the exercises in the admin dashboard

POST/admin/exercise/addExercise - Uses POST moethod to create new exercise and adds it to the database

GET/admin/exercise/viewExercise/:exerciseid - Lists a selected exercise data

GET/admin/exercise/updateExercise/:exerciseid - Page to update a selected exercise data
