Project

We are planning to build the TraceIt – Personal Digital Asset Tracker application.

Intent

We will design a structured web application that allows users to manage their purchased items, track warranty expiration dates, and organise digital assets efficiently.

  • 1st page – Dashboard showing all tracked items
  • 2nd page – Individual item details page
  • 3rd page – Add/Edit item form page
  • 4th page – User profile page

We will focus on specific areas such as database structure, route documentatioon, and user interface design.
