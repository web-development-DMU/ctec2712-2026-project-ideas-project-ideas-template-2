--Homepage--

GET/ - Displays homepage with navigation. server does no sql and renders static html

--Foods--

GET/foods - lists all foods, optionally filtered by search.

GET /foods/:foodId - shows details for a single food item.

GET /admin/foods - admin page listing foods plus a form to add new foods.

POST /foods - creates a new food item.

POST /foods/:foodId/delete - deletes a food item.

--Meals--

GET /meals - lists all meals created by the user

POST/meals - creates a new meal

GET /meals/:mealId - shows a meal and the foods inside it.

--Meal Items--

POST /meals/:mealId/items - add a food item to a meal

POST /meal_items/:itemId/update - update quantity of a food item

POST /meal_items/:itemId/delete - remove a food item from a meal

--Admin--

GET /admin/users - list all the users

POST /admin/users/:userId/delete - deletes a user (cascade deletes meals + meal_items)
