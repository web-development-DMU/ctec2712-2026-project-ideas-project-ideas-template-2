Project

We are planning to build the SkillSwap application.

Intent

SkillSwap is a peer skill exchange platform where users can list skills they offer, request skills they need, and communicate through internal messaging.

  • Homepage – Browse and search skill listings
  • Skill Page – View full details of a skill
  • Create Skill Page – Add new skill listings
  • Requests Page – Manage sent and received requests
  • Messaging Page – Communicate with other users

Each team member will contribute to the database design, route documentation, and user interface planning.
