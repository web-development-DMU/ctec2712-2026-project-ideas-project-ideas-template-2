Routes Documentation

Planned HTTP routes for SourceFlow, mapped cleanly to CRUD operations.

Core API routes

Method Route CRUD Purpose
GET/requestsReadList requests (filter by status/customer)
GET/requests/:idReadView single request + notes
POST/requestsCreateCreate a request
PUT/requests/:idUpdateUpdate request details/status
DELETE/requests/:idDeleteRemove request (and notes)
GET/requests/:id/notesReadList notes for a request
POST/requests/:id/notesCreateAdd concierge note
GET/customersReadList customers
POST/customersCreateCreate customer profile
