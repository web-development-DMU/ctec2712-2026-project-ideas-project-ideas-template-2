Application

Budget buddy

Budget Buddy is a web application designed to help users manage their personal finances. The system allows users to record income and expenses, organise transactions into categories, set monthly budgets, and track their spending habits. The goal is to provide a simple and clear way for users to monitor their financial activity and make better budgeting decisions.

About Our Team

We are a team of students developing a Budget Tracker web application to help users manage their income, expenses, and financial goals. Our system focuses on simplicity, organisation, and clear financial tracking.

    • Technical Considerations

    The Budget Buddy application will be structured as a simple CRUD web system. Users will interact with HTML forms to create, read, update, and delete financial data. Navigation will be handled using hyperlinks and anchor links to allow users to move between documentation sections easily.

    The application will include a clear separation between the user interface and the database structure. Future development will include HTTP communication between the client and server, route handling, and structured data processing.
