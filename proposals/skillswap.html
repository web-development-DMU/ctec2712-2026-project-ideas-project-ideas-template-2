SkillSwap

Peer Skill Exchange Platform

Proposed by:

  • Muhammad Ismail
  • Dennis Chinedum

Description

SkillSwap is a web application where users can list skills they offer, request skills they need, and communicate through internal messaging. The platform encourages collaboration and knowledge sharing.

Main Features

  • User accounts
  • Create and browse skill listings
  • Send and manage skill requests
  • Internal messaging between users

Database Tables

  • Users
  • Skills
  • Requests
  • Messages
