TheShopper System

Developed by Emmanuel and Yazmin, TheShopper is a personal shopping request system that allows users to submit and track sourcing requests in a structured way.

What we are building

TheShopper System allows customers to submit shopping requests and track progress, while administrators manage updates and statuses.

Users

  • Customer
  • Admin / Concierge

Core Features

  • Create request
  • Track status
  • Manage requests
