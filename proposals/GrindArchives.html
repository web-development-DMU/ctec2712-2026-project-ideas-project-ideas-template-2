Breif Description of the Application:

Grind Archives is a web application where users create a profile and curate a public gaming portfolio. Users can add games and record structured statistics such as hours played, achievements or trophies earned, completion status, platform, and personal notes. Profiles can be shared publicly, allowing other users to browse and search player archives. The platform also aims to support optional stat verification and community interaction features to enhance credibility and engagement.

A Few Paragraphs of the Application

What Problem It Solves

Many gamers track progress across multiple platforms (PlayStation, Xbox, Steam, Switch). Even when platforms show stats, they are often locked inside that platform and not presented as a single “portfolio” that’s easy to share. Grind Archives centralises that information and lets the player present it in a clean, personalised profile.

What The User Experience Is Like

A user signs up, creates a profile (bio, favourite genres, “currently grinding”, etc.), then adds games to their archive. Each game entry includes structured fields (hours played, trophies/achievements, completion %) plus optional personal notes. The user can edit entries over time, and their profile automatically reflects updates.

Why It Fits In This Module

This is a CRUD application with an HTML interface: users can Create profiles/game entries, Read (view) profiles and game archives, Update stats and details, and Delete games or accounts. The structure naturally supports multiple database tables (aiming for 3–4+ tables), which matches the project expectations.

A List of Possible Application Names

Grind Archives, TrophyShelf, PlayerVault, StatStack and Gamefolio.

Different Types of Users and What They Can Do

Standard User (Player)

  • Register / log in / log out
  • Create and edit their profile
  • Add games to their archive
  • Update game stats (hours, achievements, completion, etc.)
  • Delete a game entry
  • Choose privacy level (public profile vs private profile)

Administrator (Moderation)

  • View all users and archives
  • Remove inappropiate content (Bios/Notes/Usernames)
  • Disable accounts that break rules
  • Manage "Platform List" / "Genre List" data

Guest (Not logged in)

  • Browse/Search the games archive page
  • View public profiles
  • Cannot edit anything

Game Archive (CRUD)

  • As a user, I want to add a game to my archive so I can track it.
  • As a user, I want to edit a game entry (hours played, trophies, completion status) so the stats stay accurate.
  • As a user, I want to delete a game entry so I can remove games I no longer want listed.
  • As a user, I want to view my archive as a list and click a game to view details.

Browsing & Discovery

  • As a guest, I want to search public profiles by username so I can find people.
  • As a user, I want to filter my archive by platform/genre/status so I can find games quickly.

Admin (If Implemented)

  • As an admin, I want to remove reported content so the site stays safe and appropriate.
