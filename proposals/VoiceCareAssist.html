VoiceCare Assist

Proposed by: Mohammad Abdullah Sajid, Abdul Wahab Sajid

Brief description: VoiceCare Assist is an AI-powered voice controlled assistance system that allows users to call for help with their voice.

Paragraph Explanation: VoiceCare Assist is a web application that provides a way for users to ask for help using their voice. It is primarily designed for individuals that have limited mobility. By using a voice command, they cam activiate the system which converts their voice to text and send it as a ticket to a health care provider, who can them allocate them to a Healthcare Officer to respond to the issue.

Different Types of users:

  • PatientsCan use their voice command to activate the system. They can also use a voice command to end it
  • Healthcare ProvidersCan view and respond to the tickets created, Allocate them to a Healthcare officer, save tickets and delete tickets
  • Healthcare OfficersCan view the tickets allocated to them, respond to them

User Requirements:

  • Patients should be able to use their voice command to activate the system. They can also use a voice command to end it
  • Healthcare Providers should be able to view the tickets, modify them,
