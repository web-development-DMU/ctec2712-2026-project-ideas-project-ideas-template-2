    • Description

    TraceIt is a web application that allows users to track purchased products, store purchase information, and monitor warranty expiration dates. The system helps users organise their digital assets and avoid losing important warranty details.

    Application Features

    • Add new purchased items
    • Store purchase date and warranty expiry
    • Categorise items (Electronics, Appliances, etc.)
    • Set reminders for warranty expiration
    • Mark items as active, sold, or disposed
    • Filter items by category or status

    Database Tables

    • Users
    • Items
    • Categories
    • Reminders

    CRUD Operations

    • Create items
    • Read item details
    • Update item information
    • Delete items
